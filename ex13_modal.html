13. Modal (Dialog com Backdrop)

Modal usando <dialog> com backdrop blur e anchor positioning

Isto é um Modal

Este é um modal criado com o elemento <dialog> combinado com backdrop-filter blur para um efeito visual elegante.

Pode conter qualquer conteúdo HTML, incluindo imagens, texto, formulários e mais.

Como funciona:

<dialog id="my-popover" popover>
  <h2>Título</h2>
  <p>Conteúdo do modal</p>
  <button onclick="my-popover.close()">Fechar</button>
</dialog>

<style>
#my-popover::backdrop {
  background-color: rgba(40, 0, 80, 0.25);
  backdrop-filter: blur(3px);
}
</style>
