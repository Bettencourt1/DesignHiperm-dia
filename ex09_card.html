9. Card (Container Query)

Card que adapta layout conforme o tamanho do container, não da viewport

Título do Card

Este card usa container queries para adaptar seu layout conforme a largura disponível no container, não na janela do navegador.

Como funciona:

.card-container {
  container-type: inline-size;
}
@container (min-width: 500px) {
  .card { display: grid; grid-template-columns: 100px 1fr; }
}
@container (min-width: 700px) {
  .card { grid-template-columns: 150px 1fr; }
}
