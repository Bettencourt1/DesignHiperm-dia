4. Accordion

Usando <details> e <summary> para criar accordions

Pergunta 1: O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Fornece a estrutura e conteúdo das páginas.

Pergunta 2: O que é CSS?

CSS (Cascading Style Sheets) é usado para estilizar e fazer o layout de páginas web — por exemplo, para alterar a fonte, cor, tamanho e espaçamento do seu conteúdo.

Pergunta 3: O que é JavaScript?

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas web. Sempre que uma página faz mais do que simplesmente mostrar informações estáticas, atualiza conteúdo em tempo real, ou controla mídia multimídia.

Pergunta 4: Como funcionam os <details>?

O elemento <details> cria um widget de divulgação no qual as informações são visíveis apenas quando o widget é alternado para um estado "aberto". O elemento <summary> fornece uma legenda ou título que pode ser clicado para alternar o estado aberto/fechado.
