1. Alternar entre tema claro/escuro

Usando CSS light-dark() para adaptar cores automáticamente conforme prefers-color-scheme

Conteúdo com light-dark()

Este painel muda de cor automaticamente conforme o tema do sistema.

Exemplo de uso:

body {
  background-color: light-dark(#ffffff, #1a1a1a);
  color: light-dark(#000000, #ffffff);
}
