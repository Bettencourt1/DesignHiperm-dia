6. Badge (Anchor Positioning)

Badge posicionado usando Anchor Positioning CSS

Como funciona:

O botão define anchor-name: --inbox-button

O badge usa position: absolute com referências ao anchor via anchor()

.btn {
  anchor-name: --inbox-button;
}
.badge {
  position: absolute;
  top: anchor(--inbox-button top);
  left: anchor(--inbox-button right);
  translate: -50% -50%;
}
